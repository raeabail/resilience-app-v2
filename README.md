# resilience-app-v2
A decision-support tool that scores parcels for resiliency (or vulnerability) to five major hazards including drought, extreme temperature, riverine flooding, wildfire, and land degradation. The main goal of this tool is to turn geospatial data into actionable insight for conservation, restoration, and land management.
# Resilience App Starter v2

## Run in GitHub Codespaces
1. Open this repo in a Codespace.
2. Run task: "Start backend (FastAPI)".
3. Set port 8000 to Public.
4. Open `frontend/index.html` → enter backend URL → test.
